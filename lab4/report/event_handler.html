Обробник подій

Скрипт виконується коли відбувається певна подія.
Приклад з обробником події "keyup" 


let input_text = document.getElementById("input");
let result = document.getElementById("result_feed")
input_text.addEventListener("keyup", function () {
    result.innerText = input_text.value.toUpperCase().split("").reverse().join("")
});
